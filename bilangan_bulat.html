Apa itu Bilangan Bulat?

Bilangan bulat adalah semua kumpulan angka yang utuh (tidak punya bagian pecahan atau desimal). Angka ini sering kita pakai sehari-hari, misalnya untuk menghitung suhu, ketinggian, atau untung-rugi.

Bilangan bulat terdiri dari 3 kelompok:

  • Bilangan Bulat Positif (Bilangan Asli): Ini adalah angka yang biasa kita pakai menghitung: 1, 2, 3, 4, ... dan seterusnya.
  • Bilangan Nol: Ya, angka 0 itu sendiri!
  • Bilangan Bulat Negatif (Lawan Bilangan Asli): Ini adalah "lawan" dari bilangan positif: -1, -2, -3, -4, ... dan seterusnya.

Memakai Garis Bilangan

Cara paling gampang untuk mengerti bilangan bulat adalah pakai Garis Bilangan.

  • Angka Nol (0) ada di tengah.
  • Angka Positif (+) ada di sebelah KANAN 0. Semakin ke kanan, nilainya semakin besar.
  • Angka Negatif (-) ada di sebelah KIRI 0. Semakin ke kiri, nilainya semakin kecil.

Penjumlahan Bilangan Bulat

Saat menjumlahkan di garis bilangan, kita pakai panah. Ingat aturan mainnya ya!

  • Bilangan Positif (+) artinya panah bergerak ke KANAN.
  • Bilangan Negatif (-) artinya panah bergerak ke KIRI.

a. Positif (+) ditambah Positif (+)

Contoh: 2 + 3 = 5

Artinya: Mulai dari 0, maju (ke kanan) 2 langkah. Lalu, maju (ke kanan) lagi 3 langkah. Berhenti di angka 5!

b. Negatif (-) ditambah Negatif (-)

Contoh: -2 + (-3) = -5

Artinya: Mulai dari 0, mundur (ke kiri) 2 langkah. Lalu, mundur (ke kiri) lagi 3 langkah. Berhenti di angka -5!

c. Positif (+) ditambah Negatif (-)

Contoh: 2 + (-3) = -1

Artinya: Mulai dari 0, maju (ke kanan) 2 langkah. Lalu, mundur (ke kiri) 3 langkah. Berhenti di angka -1!

d. Negatif (-) ditambah Positif (+)

Contoh: -2 + 3 = 1

Artinya: Mulai dari 0, mundur (ke kiri) 2 langkah. Lalu, maju (ke kanan) 3 langkah. Berhenti di angka 1!

Contoh Soal Penjumlahan

Seekor katak berada di titik -5. Ia melompat ke kanan sejauh 8 langkah. Di titik berapakah katak itu sekarang?

Klik untuk melihat Pembahasan

1. Tentukan Titik Awal
Katak berada di titik -5.

2. Tentukan Pergerakan
Melompat ke kanan sejauh 8 langkah artinya ditambah positif 8 (+8).

3. Hitung Operasinya
-5 + 8

4. Bayangkan Garis Bilangan
Dari -5, bergerak 8 langkah ke kanan: -4, -3, -2, -1, 0, 1, 2, 3.

Jadi, katak itu sekarang berada di titik 3.

Sifat-sifat Penjumlahan

Penjumlahan itu punya "sifat" atau "aturan" keren yang bikin hitungan jadi gampang!

  • Sifat Komutatif (Tukar Posisi): Posisinya boleh ditukar, hasilnya tetap sama!
    Contoh: 3 + 2 sama saja dengan 2 + 3, hasilnya 5.
  • Sifat Asosiatif (Kelompok): Kamu boleh kerjakan dari mana dulu, hasilnya tetap sama!
    Contoh: (3 + 2) + 1 sama saja dengan 3 + (2 + 1).
  • Unsur Identitas (Angka Nol): Angka apapun kalau ditambah 0, hasilnya ya angka itu sendiri.
    Contoh: 3 + 0 = 3.
  • Invers (Lawan): Setiap angka punya "lawan". Jika ditambah lawannya, hasilnya pasti 0.
    Contoh: 3 + (-3) = 0.
  • Sifat Tertutup: Jika bilangan bulat ditambah bilangan bulat, hasilnya pasti bilangan bulat juga.
    Contoh: 2 + 3 = 5 (2, 3, dan 5 semuanya bilangan bulat).

Pengurangan Bilangan Bulat

Ini dia kuncinya: Mengurangi itu SAMA DENGAN Menambah dengan Lawannya.

  • Lawan dari 5 adalah -5.
  • Lawan dari -4 adalah 4.

Jadi, 2 - 3 itu sama saja dengan 2 + (-3). Gampang, kan?

a. Positif (+) dikurang Positif (+)

Contoh: 2 - 3 = -1

Sama dengan: 2 + (-3) = -1. (Maju 2, mundur 3, berhenti di -1).

b. Negatif (-) dikurang Negatif (-)

Contoh: -2 - (-3) = 1

Sama dengan: 2 + (Lawan dari -3) â†’ 2 + 3 = 5. (Maju 2, maju lagi 3, berhenti di 5).

d. Negatif (-) dikurang Positif (+)

Contoh: -2 - 3 = -5

Sifat-sifat Pengurangan

  • Pengurangan dengan Nol (0):
    • Angka apapun dikurangi 0, hasilnya angka itu sendiri. a - 0 = a (Contoh: 3 - 0 = 3).
    • Jika 0 dikurangi sebuah angka, hasilnya adalah lawan angka itu. 0 - a = -a (Contoh: 0 - 3 = -3).
  • Sifat Tertutup: Jika bilangan bulat dikurangi bilangan bulat, hasilnya pasti bilangan bulat juga.
    Contoh: 8 - 3 = 5 (8, 3, dan 5 semuanya bilangan bulat).
